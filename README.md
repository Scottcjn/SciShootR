# SciShootR

A 2D space shooter built with Unity. Originally "Starship Shooter", renamed after the associated crypto project ended.

## Features
- Space shooter gameplay with multiple levels (10+ scenes)
- Score tracking and leaderboard
- In-game shop with purchasable items
- Custom sprites and prefabs
- TextMesh Pro UI

## Tech Stack
- **Engine**: Unity 2019.4 / 2020.3
- **Language**: C#
- **Platform**: Windows (originally), cross-platform capable

## Project Structure
```
Assets/
├── BoughtAssets/     # Third-party asset packs
├── Prefabs/          # Game object prefabs
├── Scenes/           # 10+ game levels
├── ScoreBoard/       # Score tracking system
├── Scripts/          # C# game logic
├── Shop Items/       # In-game shop assets
├── SpaceShooter/     # Core shooter mechanics
├── Sprites/          # 2D sprite assets
└── TextMesh Pro/     # UI text system
```

## Building
1. Open in Unity 2020.3+
2. Open `Assets/Scenes/` and select a scene
3. Build & Run
